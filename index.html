  • Links
    • Os links, também conhecidos como âncoras, são utilizados para relacionar
      partes do mesmo documento. Por padrão, são exibidos em azul e sublinhados .
  • Como criar um link: a href="produtos.html">Produtos a
  • Links Relativos
    • Os links relativos são aqueles que apontam para páginas localizadas
      dentro do mesmo projeto. Se a página referenciada estiver na mesma pasta,
      basta mencionar o seu nome para gerar o link.
    • "a href="contato.html">Contato a
  • Links Internos
    • Os links internos permitem fazer referências a seções de sua página,
      para as quais a id é utilizada: a href="#rodape">Vá ao rodapé da página a
    • Também é possível usar como destino uma seção específica em uma página diferente: a href="contato.html#formulario">Formulario de contato a
  • Multimídia em HTML
    • Inserir imagens: São inseridas com a tag , que pertence ao grupo de tags
      que se fecham nelas mesmas (com a barra no final).
